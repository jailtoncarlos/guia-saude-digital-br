# Pesquisa com seres humanos — Lei 14.874/2024 e Inaep

Fichas de normas sobre pesquisa com seres humanos, relevantes quando o
sistema de saúde digital vier a ser utilizado em pesquisa científica
(coleta, análise, publicação com dados de participantes).

| Ficha | Norma | Tema |
|---|---|---|
| [Lei 14.874/2024](./lei-14874-2024.md) | Lei 14.874/2024 + Decreto 12.651/2025 | Nova Lei da Pesquisa com Seres Humanos; Inaep |
| [Res. CNS 738/2024](./resolucao-cns-738-2024.md) | Res. CNS 738/2024 (rep. 2025) | Banco de dados em pesquisa; rastreabilidade e retenção |

## Quando este eixo importa

- Sistema que **coleta dado diretamente** de participante de pesquisa.
- Sistema que **reutiliza dado assistencial** em projeto de pesquisa
  (mesmo que o dado tenha sido coletado para outra finalidade).
- Repositório de dados (*data lake*) que serve pesquisa.
- Dashboards que consolidam dados de participantes para estudo
  epidemiológico.

Se o sistema é **puramente assistencial** (prontuário, prescrição,
laudo), este eixo **não se aplica** — observe LGPD, SBIS e MS/DATASUS.
