## Por que a ANPD importa para saúde digital

A ANPD é o regulador da LGPD. Em 2025/2026, com o **Mapa de Temas
Prioritários** colocando **dados de saúde e especificidades do SUS** entre
os focos de fiscalização, sistemas de saúde digital passam a ter risco
regulatório elevado. Além disso, quando um projeto toca operador/provedor
fora do Brasil (cloud global, fornecedor de IA), a Resolução 19/2024
sobre transferência internacional é determinante.

O Guia Orientativo para o Poder Público (2024) é o material mais prático
para sistemas SUS: detalha bases legais, como responder a titular, como
operar DPIA em órgão público etc.
